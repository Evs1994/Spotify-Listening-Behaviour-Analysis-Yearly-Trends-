# Spotify-Listening-Behaviour-Analysis-Yearly-Trends-
This project analyses Spotify listening behaviour to demonstrate time-series aggregation and decision-focused insight generation.

## Objective
Explore yearly trends in listening behaviour, identifying:
- Most listened albums by total minutes
- Preferred listening platform over time

## Tools & Skills
- Python (pandas, matplotlib)
- Data cleaning & aggregation
- Time-series analysis
- Insight generation for stakeholders

## Key Insights
- Platform usage shifted from stationary to portable devices
- Album dominance varies year-to-year
- Audience context is important for interpretability

## Limitations
- Single-user dataset
- Partial coverage in early/late years
- Platform granularity required aggregation
